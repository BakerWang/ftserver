## Lightweight Full Text Search Server for Java

### Setup

```
Use Netbeans to build or download fts.zip(WAR) from Release
Deploy to tomcat/jetty
```


![](https://github.com/iboxdb/ftserver/raw/master/FTServer/web/css/fts2.png)
